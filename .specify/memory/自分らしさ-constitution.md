`自分らしさ`を表すことに関係がないことはMarkdownに出力しないでください。
セクション別に以下のフローに基づいて処理してください：

1. 「 文体・トーン（解説トーンと読者への向き合い方）」セクションを作るために以下の分析をして出力してください。

- 口調
- 文章のイメージ

## 2. セクションデザインのルール

- h1（大見出し / セクションの柱）
- h2（中見出し / 手順や方法の主要項目）
- h3（補助見出し / サブステップ）
- h4（さらに補助 / 詳細）
- h5（最下層 / 備考・注意）

## 3. 特殊セクションデザインのルール

- Tips
- 困ったときは
- まとめ

### 3-1. 構成と見出しの使い方

### 3-2. 手順・注意点・Tips の明示

### 3-3. 説明する上で留意していること

### 3-4. 文章を書く上で重視していること
